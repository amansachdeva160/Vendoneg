# 📖 Overview

Vendoneg is an intelligent procurement decision support prototype that demonstrates how multiple AI-inspired agents can collaborate to simplify enterprise procurement workflows.

The application guides procurement teams through supplier evaluation, quotation analysis, negotiation planning, risk assessment, savings estimation, and final procurement decision-making.

> **Note:** This project demonstrates a multi-agent procurement workflow using simulated AI reasoning. The architecture is designed for future integration with enterprise AI models and real-time procurement systems.

---

# 🎯 Problem Statement

Enterprise procurement involves multiple manual processes including:

- Comparing supplier quotations
- Evaluating vendor performance
- Planning procurement strategies
- Assessing procurement risks
- Negotiating supplier pricing
- Preparing procurement reports

These activities are often time-consuming, repetitive, and prone to inconsistent decision-making.

---

# 💡 Solution

Vendoneg provides a centralized AI-inspired procurement platform where specialized agents collaborate to support procurement professionals throughout the procurement lifecycle.

The system demonstrates how intelligent agents can assist in making procurement decisions faster, more consistently, and with better visibility.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/amansachdeva160/Vendoneg.git
```

## Navigate to Project

```bash
cd Vendoneg
```

## Install Dependencies

```bash
npm install
```

## Start Frontend

```bash
npm run dev
```

## Start Backend

```bash
cd backend
pip install -r requirements.txt
python app/main.py
```
