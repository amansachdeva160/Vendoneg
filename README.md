# 📖 Overview

Vendoneg is an intelligent procurement decision support prototype designed to simplify supplier evaluation, quotation analysis, vendor comparison, negotiation planning, and procurement decision-making.

The application demonstrates how an AI procurement assistant can help organizations make faster, smarter, and more informed purchasing decisions through an intuitive and interactive workflow.

> **Note:** The current version uses simulated AI reasoning to demonstrate the complete procurement workflow and is designed for future integration with enterprise AI models.

---

# 🎯 Problem Statement

Procurement teams often spend significant time:

- Comparing supplier quotations
- Evaluating vendor performance
- Planning negotiations
- Assessing procurement risks
- Preparing procurement reports

These processes are frequently manual, repetitive, and time-consuming.

---

# 💡 Solution

Vendoneg provides a centralized procurement platform that assists users by:

- 📄 Analyzing supplier quotations
- 🤝 Comparing vendors
- 📊 Visualizing procurement insights
- 💬 Providing negotiation suggestions
- ⚠️ Evaluating procurement risks
- 📈 Supporting procurement decision-making

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/amansachdeva160/Vendoneg.git
```

Move into the project

```bash
cd Vendoneg
```

Install frontend dependencies

```bash
npm install
```

Run frontend

```bash
npm run dev
```

Run backend

```bash
cd backend
pip install -r requirements.txt
python app/main.py
```
